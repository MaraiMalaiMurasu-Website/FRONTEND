## 🎛 How to edit each section in the admin

1. Open `https://maraimalaimurasu.com/admin` (login)
2. Sidebar → click **Home Editor**
3. Each of the 14 sections above has its own collapsible card
4. Find the section name (matches the `📍 Section N` headings above)
5. Edit any field — they auto-save to localStorage and sync to backend
6. Click **💾 Save Home Page Content** at the bottom to push to server
7. Within 10 seconds, every visitor's browser shows the new content

---

## 🔁 Resetting to PDF defaults

If you've edited fields and want to restore the May 31, 2026 PDF content:

1. Open browser DevTools (F12) → **Application** tab → **Local Storage** → `maraimalaimurasu.com`
2. Delete the key `customHomeContent`
3. Refresh the page
4. `runHomeContentMigration()` auto-applies the PDF defaults

Or simpler: in admin's Home Editor, there's an **"Apply PDF Content"** button that resets to defaults instantly.
